Статья посвящена разработке информационно-аналитической системы "Ixodes" для анализа геномов иксодовых клещей. Система позволяет хранить данные о местоположении, видовом составе и переносимых инфекциях. Она предоставляет возможность просмотра численности клещей по виду, полу, биотопу, генам и инфекциям. Встроенные алгоритмы статистики позволяют проводить сравнительный анализ встречаемости инфекций и генов, строить столбчатые и круговые диаграммы. Для расширения системы были реализованы методы кластеризации и классификации для обработки и проведения анализа набора геномов на основе их последовательностей.

Система включает в себя трехуровневую архитектуру, основанную на системе клиент-сервер. Клиентом выступает браузер пользователя, который отправляет запрос на сервер, где хранится логика приложения и база данных. На сервере выделен отдельный блок для сложных вычислений.

Для апробации методов используется разбиение штаммов на типы ВКЭ. Первый подход направлен на решение задачи генотипирования через кластеризацию на основе филогенетического дерева штаммов. Второй подход направлен на решение задачи генотипирования через классификацию.
